# Alerts

Alert system documentation.

## What Belongs Here

- Alert contract specifications
- Alert types and categories
- Alert delivery mechanisms
- Alert configuration guides
- Alert best practices

## What Does NOT Belong Here

- Alert implementation code (goes in core/sme-os/contracts/alerts.ts)
- UI components for alerts (goes in apps or packages/ui)
- Business logic for alerts (goes in core/sme-os/)
- User guides (create separate user docs if needed)
