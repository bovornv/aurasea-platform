# Language

Language and localization documentation.

## What Belongs Here

- Language strategy and guidelines
- Translation workflows
- Locale configuration guides
- Typography guidelines for Thai/English
- RTL/LTR considerations

## What Does NOT Belong Here

- Translation files (goes in packages/i18n/)
- i18n implementation code (goes in packages/i18n/)
- UI components (goes in packages/ui/)
- Application-specific translations (goes in apps/)
