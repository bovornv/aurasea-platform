# Labor Domain

Labor planning and optimization domain logic.

## Responsibilities

- Labor demand forecasting
- Schedule optimization
- Labor cost analysis
- Capacity planning

## Boundaries

- **DO NOT** reference specific job roles (e.g., "waiter", "housekeeper")
- **DO NOT** include vertical-specific labor concepts
- **DO** work with generic labor units and time periods
- **DO** provide abstract labor planning models
