# UI Package

Shared UI components and design system.

## Responsibilities

- Provide reusable UI components
- Maintain design system (Stripe + Linear style)
- Define typography, colors, and layout utilities
- Support desktop-first responsive design

## Boundaries

- **DO NOT** include business logic
- **DO NOT** include vertical-specific components
- **DO** provide generic, reusable components
- **DO** maintain design consistency across apps

## Structure

- `components/` - Reusable UI components
- `typography/` - Typography system
- `colors/` - Color palette and tokens
- `layout/` - Layout utilities and components
